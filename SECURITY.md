# Security policy

## Scope

This is an independent static redesign demo, not Mault's production website. Do not report issues with `mault.ai` here.

## Reporting a vulnerability

Please report vulnerabilities privately to `security@fltechadventures.com`. Include the affected route or file, reproduction steps, impact, and any suggested remediation. Do not open a public issue containing exploit details or secrets.

## Supported version

Only the latest commit on `main` is supported. The hosted preview is intentionally isolated, read-only, and excluded from indexing.
